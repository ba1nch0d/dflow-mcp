Quick Start

Claude Desktop

Add to Claude Desktop configuration for immediate access to prediction market data.

"dflow-mcp": {
  "command": "bun",
  "args": ["run", "./src/index.ts"]
}

Terminal Usage

Run locally with Bun for development and testing.

git clone repo
cd dflow-mcp
bun install
bun run dev

Cloud Deploy

Deploy to Netlify or any static hosting platform.

netlify deploy --prod
# or connect repo to
# Netlify for auto-deploy
